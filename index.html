Static preview tool

Jinja newsletter preview

Paste a Jinja or HTML email template on the left and render a live preview on the right. This app is static, so it can be hosted on GitHub Pages. It uses Nunjucks in the browser, which is very close to Jinja2 for common newsletter templates.
